## 1. Dependencies

- Install **Python 3**
- Install **Pip**


## 2. Install env
- pip install virtualenv

## 3. Create env

```bash
`python3 -m venv env
````

## Activate env

```bash
source env/bin/activate
``` 

```bash
.\env\Scripts\activate # windows
```

## Install dependencies

```
pip install -r requirements.txt
``` 

## Start

```
python3 app.py
```
